## Что вообще за объект? Что означает равенство объектов?

Объект «Спортсмен» (Athlete) — это конкретный человек, занимающийся спортом, обладающий уникальным набором физических характеристик (имя, вес, рост) и личным рекордом. Равенство объектов определяется по их основным характеристикам: два спортсмена считаются равными, если у них совпадают имя, вес, рост и личный рекорд. Состояния (активность, здоровье, уровень подготовки, мораль) при сравнении не учитываются, так как они могут изменяться во времени, но спортсмен остаётся тем же лицом. Это реализовано в методе __eq__:

```py
def __eq__(self, other):
    if not isinstance(other, Athlete):
        return False
    return (self._name == other._name and
            self._weight == other._weight and
            self._height == other._height and
            self._personal_record == other._personal_record)

```

## Есть ли состояние?

объект Athlete имеет несколько логических состояний, которые влияют на его поведение:

| Состояние | Возможные значения | Влияние на поведение |
|------------------|-------------------------------------------------|--------------------------------------------------------------------------------------|
| is_active | True / False | Если False, методы изменения рекорда (set_record) вызывают исключение RuntimeError |
| health_status | "healthy", "injured", "recovering" | При "injured" изменение рекорда запрещено; при "recovering" выдаётся предупреждение |
| training_level | "beginner", "intermediate", "advanced", "professional" | Ограничивает максимально допустимое значение личного рекорда |
| morale | целое число от 0 до 10 | В текущей реализации не влияет, но может использоваться для расширения логики |

Все эти состояния могут изменяться через специальные методы (activate, deactivate, injure, recover, heal, set_training_level, а также сеттер morale). Поведение объекта зависит от текущих значений состояний.
